# 사용자 시나리오 (User Scenarios) - 마이링크 (MyLink)

> **작성 규칙**: "사용자는 [목적]하기 위해 [행동]한다." 형식 기준

---

## 1. 방문자 (Visitor) 시나리오: 마이링크 페이지 방문 과정

- 사용자는 **소유자의 다양한 활동 정보와 포트폴리오를 한 곳에서 확인하기 위해** 제공된 고유 URL(예: `mylink.io/@jiyun`)을 클릭하여 마이링크 퍼블릭 페이지에 접속한다.
- 사용자는 **소유자가 운영하는 주요 SNS 채널(인스타그램, 깃허브 등)로 바로 이동하기 위해** 페이지 상단 프로필 영역에 위치한 '소셜 아이콘'을 클릭한다.
- 사용자는 **소유자가 강조하고 싶은 특정 웹사이트나 콘텐츠를 열람하기 위해** 화면 중앙에 나열된 여러 개의 '링크 버튼(카드)' 중 하나를 선택하여 클릭한다.
- 사용자는 **소유자의 마이링크 프로필을 지인에게 전달하기 위해** 페이지 우측 상단의 '공유하기' 아이콘을 누르고 'URL 복사'를 선택한다.

---

## 2. 소유자 (Owner) 시나리오: 링크 관리(추가, 수정, 삭제) 과정

### 2.1 사이트 접속 및 링크 추가 과정
- 사용자는 **자신의 마이링크 프로필과 링크 목록을 관리하기 위해** 사이트 메인에서 소셜 로그인(OAuth)을 통해 관리자 대시보드에 접속한다.
- 사용자는 **사용자들에게 공유할 새로운 웹페이지 주소를 등록하기 위해** 대시보드의 메인 화면에 있는 '+ 새 링크 추가' 버튼을 클릭한다.
- 사용자는 **방문자가 해당 링크가 어떤 내용인지 직관적으로 알게 하기 위해** 생성된 빈 카드 영역에 'URL 주소'와 버튼에 표시될 '타이틀 글귀'를 입력한다.
- 사용자는 **작성된 링크를 자신의 퍼블릭 페이지에 즉시 노출하기 위해** 저장을 완료한다. (오토 세이브 방식 또는 저장 버튼 클릭)

### 2.2 링크 정보 수정 및 상태 제어 과정
- 사용자는 **기존에 입력해 둔 URL 주소가 변경되거나 타이틀의 오타를 바로잡기 위해** 대시보드 내 해당 링크 카드의 '수정' 영역(또는 연필 아이콘)을 클릭해 텍스트를 고친다.
- 사용자는 **특정 이벤트 기한이 종료되어 해당 링크를 잠시 숨기기 위해** 링크 카드의 우측에 있는 활성화/비활성화 스위치(Toggle)를 눌러 비활성(Off) 상태로 변경한다.
- 사용자는 **자신의 퍼블릭 페이지에서 가장 중요하게 보여줄 링크를 상단에 배치하기 위해** 대시보드 링크 카드의 좌측 이동 핸들을 마우스로 잡고 위아래로 끌어다 놓아(Drag & Drop) 순서를 변경한다.

### 2.3 링크 삭제 과정
- 사용자는 **더 이상 유효하지 않거나 영구적으로 필요가 없어진 링크 정보를 대시보드에서 지우기 위해** 해당 링크 하단(혹은 우측)에 있는 '삭제(휴지통 아이콘)' 버튼을 클릭한다.
- 사용자는 **실수로 링크를 지우는 것을 방지하기 위해** 화면에 표시된 "정말 이 링크를 삭제하시겠습니까?"라는 경고 팝업을 확인한 뒤, 최종적으로 '확인' 버튼을 더블 클릭 혹은 단순 클릭하여 연동을 해제한다.
