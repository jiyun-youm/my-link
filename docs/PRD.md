# 제품 요구사항 정의서 (PRD) - 마이링크 (MyLink)

## 1. 프로젝트 개요 (Project Overview)

- **프로젝트명**: 마이링크 (MyLink)
- **목적**: 흩어져 있는 개인의 다양한 링크(SNS, 포트폴리오, 블로그, 쇼핑몰 등)를 단 하나의 페이지로 통합하여 직관적이고 간편하게 공유할 수 있는 '나만의 멀티 링크 프로필' 서비스 구축.
- **대상 사용자 (Target Users)**:
  - **인플루언서 및 크리에이터**: 인스타그램, 유튜브, 틱톡 등 프로필 링크 제한이 있는 플랫폼에서 다양한 외부 링크를 안내하고 싶은 사용자.
  - **프리랜서 및 구직자**: 온라인 포트폴리오, 이력서, 링크드인, 개인 블로그 등을 한곳에 모아 명함처럼 활용하고 싶은 사용자.
  - **소상공인 및 비즈니스 운영자**: 온라인 스토어, 카카오톡 문의, 예약 페이지 등을 고객에게 원클릭으로 제공하고 싶은 사용자.
  - **일반 대중**: 취향이나 관심사가 담긴 링크들을 지인들과 손쉽게 공유하고 싶은 모든 사용자.

---

## 2. 핵심 기능 목록 (Core Features)

### 필수 기능 (MVP - Minimum Viable Product)
향후 가장 먼저 개발되어야 할 핵심 단위입니다.
1. 소셜 로그인 기반 회원가입 및 인증
2. 고유 프로필 및 사용자 정보 관리
3. 단일 링크 및 소셜 링크 관리 보드
4. 퍼블릭 멀티 링크 페이지 뷰
5. 기본 테마 및 디자인 설정

### 선택/확장 기능 (Optional / Future Features)
MVP 런칭 이후 사용자 참여를 높이고 수익화를 도모할 수 있는 추가 기능입니다.
6. 고급 디자인 커스터마이징 기능
7. 방문자 및 링크 클릭 통계 (Analytics)
8. 이메일/구독자 수집 폼 통합
9. 커스텀 도메인(개인 도메인) 연결

## 3. 핵심 기능 상세 설명 (Feature Details)

### [필수 기능]

**1. 회원가입 및 계정 관리**
- **소셜 로그인 전용 인증(OAuth)**: 사용자의 빠르고 간편한 접근을 위해 Google, Kakao, GitHub 등의 소셜 로그인 연동 방식을 사용. (일반 이메일/비밀번호 가입 폼 생략)
- **고유 URL 생성**: 가입 시 원하는 아이디를 입력받아 본인만의 고유 링크를 생성함 (예: `mylink.io/@사용자ID`).
- **계정 설정 및 관리**: 프로필 기본 정보 수정(닉네임 등) 및 소셜 연동 해제/계정 탈퇴 기능.

**2. 프로필 관리 기능**
- **프로필 이미지**: 프로필 사진 업로드, 변경, 삭제 기능 구현. 사이즈 최적화 적용.
- **자기소개(Bio)**: 프로필 하단에 표시될 이름(Display Name)과 짧은 자기소개 텍스트 입력 기능.
- **소셜 아이콘 세트**: 인스타그램, 트위터, 유튜브 등 주요 SNS는 버튼이 아닌 상단 아이콘 형태로 별도 노출할 수 있는 기능.

**3. 링크 관리 보드 (대시보드)**
- **링크 추가/수정/삭제**: 새로운 링크 버튼 생성을 위한 URL 및 타이틀 텍스트 입력. 
- **드래그 앤 드롭 (Drag & Drop)**: 사용자가 시각적으로 직접 링크 카드의 순서를 변경할 수 있도록 지원.
- **링크 토글 버튼**: 링크 정보를 삭제하지 않고도 퍼블릭 페이지에서 임시로 보이지 않게 켜고 끄는(Show/Hide) 활성화 토글 탑재.

**4. 퍼블릭 멀티 링크 페이지 뷰**
- **반응형 웹 UI (Responsive)**: 공유된 링크를 모바일, 태블릿, 데스크탑 등 사용자의 디바이스 환경에 맞춰 완벽하게 보여주는 뷰 페이지. 
- **공유 기능**: 프로필 URL 복사 및 QR 코드 생성 기능 지원.

**5. 기본 테마 및 디자인 설정**
- **테마 템플릿**: Light, Dark, Pastel 등 사전 정의된 몇 가지 디자인 테마 제공.
- **버튼 스타일링**: 버튼의 테두리 모양(직각 사각형, 둥근 모서리, 완전히 둥근 알약 형태) 선택 옵션 제공.

### [선택/확장 기능]

**6. 방문자 및 링크 클릭 통계 (Analytics)**
- 최근 7일/30일간의 프로필 방문자 수 표시.
- 각 링크별 총 클릭 수 통계 및 클릭률(CTR) 계산 대시보드 제공.
- 트래픽의 유입 경로(Referral) 차트화.

**7. 고급 디자인 커스터마이징**
- 배경 요소에 솔리드 컬러 외에도 그라디언트, 패턴, 커스텀 이미지 업로드 지원.
- 외부 폰트(Google Fonts) 선택 기능 및 버튼 호버 애니메이션 커스텀 옵션 도입.
